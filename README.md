# CURATED

A modern and minimalist image gallery built with HTML, CSS, and JavaScript. CURATED showcases a carefully selected collection of high-quality images with a clean user interface and smooth browsing experience.

## ✨ Features

* Minimalist and responsive design
* Elegant landing page
* Dedicated gallery section
* Smooth navigation
* Clean typography and modern layout
* Optimized image presentation
* Mobile-friendly interface

## 📸 Preview

CURATED provides a distraction-free viewing experience, allowing users to browse a curated collection of beautiful images in a simple yet visually appealing interface.

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript

## 🚀 Getting Started

1. Clone the repository.

```bash
git clone https://github.com/yourusername/curated.git
```

2. Navigate into the project folder.

```bash
cd curated
```

3. Open `index.html` in your browser or use a live development server such as the VS Code Live Server extension.
